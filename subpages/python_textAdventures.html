Text Adventures

I've created a few text adventures in Python.
Lost in the Woods is my most recent creation, and I made an effort to design it in a smarter way - separating the story from the "engine" (see source code) - and also used curses to develop the command line interface.

My previous attempt, Only One Day, was slightly less sophisticated - story options were built in directly, however I used a data structure to classify the different endings. This one was made as a submission to the GMTK Game Jam 2019, recieving 27 ratings with a 3.4/5 overall, as well as 15 positve reviews/comments.

Both of these can be found on my Itch.io profile.
I've also included the source code to my very first attempt (Dungeon) in my GitHub old-projects repository.
