Spotify API

I am an avid Spotify user, so I developed a flask web-app which used the Spotify Web API to display album art (or playlist art) and a progress bar of the currently playing track. This can run on my laptop in the background (see screenshots below), but for something more interactive I got the server and webpage running and displaying on a Raspberry Pi Zero - using a 3.5" touchscreen hat interfacing with the GPIO pins - to sit on my desk (see right).
